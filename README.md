# MedSync – Remote Patient Vitals Monitor

This project contains two modules:

## 1. Clinical Data Processor
Java Spring Boot backend for receiving patient vitals, checking thresholds, and storing records.

## 2. PatientVitalsSimulator
Python simulator that generates patient vitals and sends them to the Java backend.

## Tech Stack
- Java
- Spring Boot
- H2 Database
- Python
- REST API
